Selected personnal projects

Some of the projects I made during the past 5 years.

Ray Tracing Based Renderer
I implemented a small framwork allowing for simple rendering tasks. This is part of my SPBL2 library. While it is not a production grade renderer, it's able to render scenes quite efficiently. I made use of great reference books, courses and projects such as: Nori, PBRT, AppleSeed and Embree. The Glass dragon picture shown at the top of this page is made using this software.
SPLB2
SPLB2 is a multiplatform, small to moderate size (190k~ LOC insertions/deletions for 600 commits as of August 12 2021) C++14, multipurpose library. This is basically a container for every interesting feature, idea, concept, algorithm I stumbled upon and had time to implement. This serves as training and reference for future my projects. It contains a variety of implementation of not well documented containers and data structures, message passing/serialization tools (similar to Blizzard's JAM), a small BLAS, some thread pools, memory allocators, CPU intrinsic wrappers, file formats readers, signal processing tools, relational database relation optimizer, academic security exploit implementation (to learn the basics and understand the risks) and rendering acceleration structure. For now it's opensource but not "open contribution", contact me if you want to get access.
Shader Player
This is a toy project that I made back in 2016 after discovering the ShaderToy website made by Inigo Quilez. It showed me the beautiful things you can do using maths and a computer. I wanted to make my own "shader player" too and it served as a pretext to learn a little bit about OpenGL. This project is deprecated and does not live up to my current coding standards, still, it's functional and can be used to showcase ShaderToy productions some of which are very pretty. Sources here.
Other
